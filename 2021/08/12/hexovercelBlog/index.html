如何使用Hexo和Vercel做出博客

发表于 2021-08-12 更新于 2021-08-16 分类于 Web 阅读次数：

准备工作

  • GitHub

    我们需要注册一个GitHub账号并登录

  • Vercel

    我们需要使用GitHub账号来登录Vercel

  • Node.js

    我们需要下载一个合适版本的Node.js

  • 最后

    我们需要Win+R打开“运行”，输入cmd点击确定。在cmd里依次输入

    node -v
    npm -v

    后回车，如若cmd报错请重新安装node.js

正式开始

  • 下载Hexo

    请在cmd里输入

    npm install hexo-cli -g

    并回车

    等待一会后可以在cmd输入

    hexo -v

    回车后等待一会如果没有返回一长串版本号请重新执行

    npm install hexo-cli -g

    并回车

  • Hexo初始化

    请在桌面创建一个文件夹，取名“blog”，并在cmd执行

    cd Desktop\blog

    后执行

    hexo init

  • 将博客上载到Vercel

    等待下载完博客文件后进入文件夹，点击最上方的地址栏（在两个箭头和搜索栏中间的地址栏），输入

    cmd

    并回车，执行

    vercel

    并回车，这时可能会要求你登录，按照操作就行了。还会问你各种问题，比如项目名称之类的（我只记得第一个了，第一个请选择Y）。接着在返回内容结尾出现

    run “vercel –prod”

    时请执行（如果没有出现以上字段但是给你了个链接就不需要再执行以下命令了！直接打开链接即可）

    vercel –prod

    继续等一会时你就可以复制系统给你的链接进入浏览器查看你的博客了。进入Vercel官网还可以编辑免费域名或是你购买的域名了~

  • 如何写作

    Win+R打开“运行”，输入cmd点击确定。在cmd输入

    hexo new 替换成你自己的文章标题

    后回车，进入C:\Users\Administrator\Desktop\blog\source\posts找到你创建的文章，下载Microsoft Visual Studio Code后使用它打开你的文章使用Markdown语法进行编辑即可。

  • 更新我的博客

    进入blog文件夹，点击最上方的地址栏（在两个箭头和搜索栏中间的地址栏），输入

    cmd

    并回车，执行

    vercel

    并回车，按照操作就行了（我只记得第一个了，第一个请选择Y）。接着在返回内容结尾出现

    run “vercel –prod”

    时请执行

    vercel –prod

    之后即可进入网站看看你的文章了
